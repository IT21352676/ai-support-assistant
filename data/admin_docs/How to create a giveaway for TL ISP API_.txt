    - This guide will help you to create a one-time 100% coupon to give free products to customers.
    - After the first month, the customers will get charged the full amount.
    - To prevent customers from abusing the coupons, the coupons should be deleted after they are used.
      - Please go to the product tab of the stripe
      - Select coupon
      - Select the required product and fill in the relevant details
      - Select the type - "Percentage Discount" & make sure to enter 100% as the percentage off so that a free product to be created for one-time.
    - You will be charged even if this is given for free. Hence regardless of the price paid, you will be billed for the amount you have created for the giveaway.
    - 
      - Select the duration as
      - Once
      - & keep the redemption list unticked.
